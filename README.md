<h1 align="center">💰 Expense Tracker API</h1>

<p align="center">
A RESTful API built with <b>FastAPI</b> that allows users to securely track and manage their expenses.
</p>

<hr>

<h2>📌 Project Overview</h2>

<p>
This project is a backend API for an expense tracking system. 
It allows users to register, authenticate, and manage their personal expenses securely.
</p>

<p>
The project demonstrates important backend concepts such as:
</p>

<ul>
<li>REST API development</li>
<li>JWT Authentication</li>
<li>Database management using SQLAlchemy</li>
<li>Secure password hashing</li>
<li>CRUD operations</li>
</ul>

<hr>

<h2>🚀 Features</h2>

<ul>
<li>👤 User Registration</li>
<li>🔐 JWT-based Login Authentication</li>
<li>💸 Create, Read, Update, and Delete Expenses</li>
<li>🛡 Password Hashing for security</li>
<li>📦 Database storage using SQLAlchemy</li>
<li>📚 Interactive API documentation with Swagger</li>
</ul>

<hr>

<h2>🛠 Tech Stack</h2>

<ul>
<li><b>Backend:</b> FastAPI</li>
<li><b>Language:</b> Python</li>
<li><b>Database:</b> PostgreSQL / SQLite</li>
<li><b>ORM:</b> SQLAlchemy</li>
<li><b>Authentication:</b> JWT (JSON Web Token)</li>
<li><b>Validation:</b> Pydantic</li>
</ul>
